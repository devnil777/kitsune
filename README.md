# Кицунэ

Обратный отсчёт до контрольных точек дня. Добавляйте время встреч, созвонов, дедлайнов — приложение покажет точный отсчёт до каждой из них.

## Установка на телефон

1. Откройте [страницу проекта](https://devnil777.github.io/kitsune/) в Chrome или Samsung Internet
2. Откройте меню браузера → «Добавить на главный экран»
3. После установки работает офлайн благодаря service worker

## Как использовать

- **Добавить точку**: откройте вкладку «Кицунэ» → «Добавить время» либо вкладку «Настройки»
- **Выбрать цель**: нажмите на точку под кольцом или сделайте вертикальный свайп на экране отсчёта
- **Напоминания**: включите уведомления в настройках — приложение напомнит в фоне
- **Синхронизация**: войдите через Google, чтобы сохранить точки между устройствами
